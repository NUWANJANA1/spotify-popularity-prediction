# 🎵 Spotify Track Popularity Prediction

Predicting Spotify track popularity using audio features and Machine Learning.

## Dataset
- Source: Kaggle — Spotify Tracks Dataset
- Records: ~114,000 tracks

## Models Used
- Linear Regression
- Random Forest Regressor ✅ Best (R² = 0.763)

## Tools
Python | Pandas | Scikit-learn | Matplotlib | Seaborn

## Key Findings
- Energy is the strongest predictor (38% importance)
- Danceability is second (27%)
- K-Pop and Pop genres have highest average popularity
